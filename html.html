O que é HTML?

linguagem de marcação padrão utilizada para criar páginas web. Ele define a estrutura do conteúdo na internet, usando uma série de "tags" ou "marcadores" para organizar e formatar textos, imagens, links, tabelas, formulários e outros elementos dentro de uma página.

<html></html>: Marca o início de um documento HTML

<head></head>: Contém metadados sobre a página, como título, links de CSS e scripts

<meta>: Fornece metadados como informações sobre o charset (conjunto de caracteres que define como os textos são representados e codificados em sistemas de computação.), autor da página, e configurações de visualização.
